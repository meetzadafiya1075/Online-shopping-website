IWP Online Store Project
Overview

This project is a simple online shopping website created as part of the Internet and Web Programming (IWP) assignment.
It consists of multiple HTML and CSS files simulating an e-commerce platform with separate sections for men, women, and kids, along with pages for login, product ordering, and confirmation.

Features

Home Page: Overview of the website with navigation links.

Login Page: Simple user login interface.

Category Pages: Separate sections for Men, Women, and Kids products.

Order Page: Form for users to place orders.

Confirmation Page: Displays the final order summary.

Custom CSS: Each section styled individually using linked CSS files.

File Structure
iwp assignment.css      → Main stylesheet for common styling
iwp login css.css       → CSS for login page
iwp end.css             → CSS for order confirmation page
iwp home.html           → Home page
iwp login.html          → Login form
iwp men.html            → Men’s section
iwp women.html          → Women’s section
iwp kids.html           → Kids’ section
iwp order.html          → Order placement page
iwp create.html         → Product creation/registration page
iwp end.html            → Order confirmation page

How to Run

Extract the ZIP file.

Open iwp home.html in any web browser.

Navigate between pages using the menu or links provided.

Technologies Used

HTML5 for structure

CSS3 for styling
